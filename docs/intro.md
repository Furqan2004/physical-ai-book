# Welcome to Physical AI & Humanoid Robotics

**From Digital Intelligence to Embodied Robots**

---

## 📖 What is This Book?

This is a comprehensive **13-week crash course** on Physical AI and Humanoid Robotics — a free, online textbook designed for students and developers who want to learn how to build intelligent robots that can perceive, think, and act in the physical world.

Unlike traditional AI that operates purely in software, **Physical AI** (also called Embodied AI) integrates intelligence into physical systems, enabling robots to interact with the real world through sensors, actuators, and machine learning.

---

## 🎯 What Will You Learn?

By completing this course, you will gain expertise in:

| Part | Topic | What You'll Learn |
|------|-------|-------------------|
| **Part 1** | Foundations of Physical AI | Understand what Physical AI is, how it differs from digital AI, and the landscape of humanoid robotics in 2024 |
| **Part 2** | ROS 2 Robotics | Master the Robot Operating System (ROS 2) — nodes, topics, services, actions, and building ROS 2 packages |
| **Part 3** | Digital Twin Simulation | Learn Gazebo and Unity for robot simulation, sensor modeling, and virtual testing |
| **Part 4** | NVIDIA Isaac Platform | Explore Isaac Sim, Omniverse, perception systems, and sim-to-real transfer |
| **Part 5** | Humanoid Robot Development | Study kinematics, locomotion, manipulation, and human-robot interaction |
| **Part 6** | Vision-Language-Action | Build autonomous humanoids with VLA models, LLM planning, and voice commands |

## 📋 Prerequisites

Before starting this course, you should have:

### Required Knowledge
- **Basic Programming**: Familiarity with Python (variables, functions, loops, classes)
- **High School Math**: Algebra, basic trigonometry, and understanding of coordinates
- **Computer Literacy**: Comfortable with command line, file systems, and Git basics

### Recommended (But Not Required)
- **Linear Algebra**: Vectors, matrices, transformations
- **Physics Basics**: Understanding of forces, motion, and gravity
- **AI/ML Concepts**: Basic awareness of machine learning terminology

> **💡 Tip**: If you're missing some prerequisites, don't worry! Each chapter includes background explanations and references to catch up.

## 📚 How to Use This Book

### Course Structure
- **13 Weeks**: Each week represents a chapter covering specific topics
- **6 Parts**: Chapters are grouped into logical parts
- **Self-Paced**: Study at your own speed — skip ahead or review as needed

### Each Chapter Contains
- 🎯 **Learning Objectives**: What you'll understand after reading
- 📖 **Theory**: Verified content with academic and industry references
- 💻 **Code Examples**: Practical implementations you can run
- 🔧 **Exercises**: Hands-on activities to reinforce learning
- 📝 **Summary**: Key takeaways
- 📚 **References**: Authoritative sources for deeper study

### Recommended Workflow
1. **Read** the chapter theory and examples
2. **Run** the code examples yourself
3. **Complete** the practical exercises
4. **Review** the references for deeper understanding
5. **Move** to the next chapter
